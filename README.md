# WEAPP_API

An flask based rest-ful api for cross-platform user using wechat in IOS and Android.

## Version

- Beta2.0(current)
- Beta1.0

## Features

### User System

- SMS
- openid
- auth

### Role System

- Role_A
- Role_B

### Event System

- Create
- Interact
- Finish

## Data Stream

- Name
- Gender
- Age
- BirthDay
- Country
- Email
- Cell
- Avatar
- Photo
- Audio
