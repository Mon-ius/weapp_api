### Deploy

#### 1. Create New virtual env (OPTIONAL)

- For Anaconda
  > conda create -n api python=3.6
  > source activate api

- For virtualenv
  > virtualenv -p /usr/bin/python3.6  api
  > . api/bin/activate

#### 2. Install requrements

> pip install -r requirements-dev.txt

#### 3. Run

> python ./run.py
